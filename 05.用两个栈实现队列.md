# 用两个栈实现队列
## 题目描述
用两个栈来实现一个队列，完成队列的Push和Pop操作。 队列中的元素为int类型。
## 解题思路
### 两个栈实现队列，那么要符合队列的先进先出的规则，我们需要用两个栈，并要考虑到栈是否为空的条件。
```
class Solution:
    def __init__(self):
        self.Stack1=[]
        self.Stack2=[]
    def push(self, node):
        # write code here
        self.Stack1.append(node)
    def pop(self):
        # return xx
         if self.Stack2!=[]:
                return self.Stack2.pop()
         while self.Stack1:
             self.Stack2.append(self.Stack1.pop())
         return self.Stack2.pop()
```
