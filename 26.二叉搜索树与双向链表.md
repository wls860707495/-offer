# 二叉搜索树与双向链表
## 题目描述
输入一棵二叉搜索树，将该二叉搜索树转换成一个排序的双向链表。要求不能创建任何新的结点，只能调整树中结点指针的指向。
## 解题思路
### 非递归，利用两个栈，先利用栈将二叉搜索树中所有的节点按照中序遍历进行存储，之后按照顺序将节点取出组成一个中序链表
```
# -*- coding:utf-8 -*-
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None
class Solution:
    def Convert(self, pRootOfTree):
        # write code here
        if not pRootOfTree:
            return None
        p = pRootOfTree
        stack = []
        restack = []
        while p or stack:                  #按照中序遍历将节点存储以保证其有序
            if p:
                stack.append(p)
                p = p.left
            else:
                node = stack.pop()
                restack.append(node)
                p = node.right
                
        Res = restack[0]               #便于输出整个双向链表 
        while restack:
            top = restack.pop(0)
            if restack:
                top.right = restack[0]
                restack[0].left = top
        return Res
```
### 递归
```
# -*- coding:utf-8 -*-
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None
class Solution:
    def Convert(self, root):
        if not root:
            return None
        if not root.left and not root.right:
            return root          
        left = self.Convert(root.left)            # 将左子树构建成双链表，返回链表头
        p = left 
        while left and p.right:                        # 定位至左子树的最右的一个结点
            p = p.right
        if left:                                # 如果左子树不为空，将当前root加到左子树链表
            p.right = root
            root.left = p
        right = self.Convert(root.right)                 # 将右子树构造成双链表，返回链表头
        if right:
            right.left = root
            root.right = right    
        return left if left else root
```
