# 二叉树的镜像
## 问题描述
操作给定的二叉树，将其变换为源二叉树的镜像。
## 解决方法
### 递归的思想。判断空树以及根节点是否有子节点。从根节点开始，若有子节点，则左右交换，再使用递归将左右子树的节点也进行交换。
```
# -*- coding:utf-8 -*-
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None
class Solution:
    # 返回镜像树的根节点
    def Mirror(self, root):
        # write code here
        if root == None:
            return 
        elif root.left==None and root.right==None:
            return 
        else:
            tmp = root.left
            root.left = root.right
            root.right = tmp
            self.Mirror(root.left)
            self.Mirror(root.right)
            return 
```
### 非递归，使用栈的思想
```
# -*- coding:utf-8 -*-
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None
class Solution:
    # 返回镜像树的根节点
    def Mirror(self, root):
        # write code here
        if root == None:
            return 
        if root.left==None and root.right==None:
            return 
        stack = []
        stack.append(root)
        while stack != None:
            tree = stack.pop()
            if tree.left!=None or tree.right!=None:
                tmp = tree.left
                tree.left = tree.right
                tree.right = tmp
            if tree.right != None:
                stack.append(tree.right)
            if tree.left != None:
                stack.append(tree.left)
                    
                  
```
