# 反转链表
## 题目描述
输入一个链表，反转链表后，输出新链表的表头
## 解题思路
### 将链表原地逆置，后直接输出即可(定义了三个指针，较为复杂)
```
# -*- coding:utf-8 -*-
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None
class Solution:
    # 返回ListNode
    def ReverseList(self, pHead):
        # write code here
        if pHead == None or pHead.next == None:
            return pHead
        pre = pHead
        mid = pre.next
        last = mid.next
        pHead.next = None
        while last != None:
            mid.next = pre
            pre = mid
            mid = last
            last = last.next
        mid.next = pre
        return mid
```
### 简便写法(亦为原地逆置)
```
# -*- coding:utf-8 -*-
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None
class Solution:
    # 返回ListNode
    def ReverseList(self, pHead):
        # write code here
        if pHead == None or pHead.next == None:
            return pHead
        pre = None
        mid = pHead
        while mid != None:
            tmp = mid.next
            mid.next = pre
            pre = mid
            mid = tmp
        return pre
```
## 总结
注意链表最后的节点
