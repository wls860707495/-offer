# 从上往下打印二叉树
## 题目描述
从上往下打印出二叉树的每个节点，同层节点从左至右打印。
## 解题思路
### 借助队列对二叉树中的元素进行遍历，并将所有二叉树节点的值放入list中进行返回
```
# -*- coding:utf-8 -*-
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None
class Solution:
    # 返回从上到下每个节点值列表，例：[1,2,3]
    def PrintFromTopToBottom(self, root):
        # write code here
        str = []
        list = []
        if root == None:
            return list
        else:
            str.append(root)
            while str:
                if str[0].left != None:
                    str.append(str[0].left)
                if str[0].right != None:
                    str.append(str[0].right)
                list.append(str.pop(0).val)
            return list
            
            
```
