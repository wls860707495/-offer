# 数字在排序数组中出现的次数
## 题目描述
统计一个数字在排序数组中出现的次数。
## 解题思路
### 循环顺序统计
```
# -*- coding:utf-8 -*-
class Solution:
    def GetNumberOfK(self, data, k):
        # write code here
        if len(data) <= 0:
            return 0
        count = 0
        num = 0
        for i in range(len(data)):
            if data[i] == k:
                num = i
                break
        for j in range(num,len(data)):      #因为是排序数组，所以连续统计完后便跳出循环，省去多余操作。
            if data[j] == k:
                count += 1
            else:
                break
        return count  
```
### 使用二分法(有序数组，使用效率会更高)
```
# -*- coding:utf-8 -*-
class Solution:
    def GetNumberOfK(self, data, k):
        # write code here
        if len(data) <= 0:
            return 0
        length = len(data)
        index = self.Binarysearch(k,data,length)
        if index == -1:
            return 0
        count = 1
        for i in range(1,length):
            if index+i < length and data[index+i] == k:
                count += 1
            if index-i >= 0 and data[index-i] == k:
                count += 1
        return count    
    def Binarysearch(self,k,data,length):
        start = 0
        end = length-1
        while start<=end:
            mid = (start + end)/2
            if data[mid] < k:
                start = mid +1
            elif data[mid] > k:
                end = mid -1
            else:
                return mid
        return -1       
```
