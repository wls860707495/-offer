# 数字在排序数组中出现的次数
## 题目描述
统计一个数字在排序数组中出现的次数。
## 解题思路
### 循环顺序统计
```
# -*- coding:utf-8 -*-
class Solution:
    def GetNumberOfK(self, data, k):
        # write code here
        if len(data) <= 0:
            return 0
        count = 0
        num = 0
        for i in range(len(data)):
            if data[i] == k:
                num = i
                break
        for j in range(num,len(data)):      #因为是排序数组，所以连续统计完后便跳出循环，省去多余操作。
            if data[j] == k:
                count += 1
            else:
                break
        return count  
```
