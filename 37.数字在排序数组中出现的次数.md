# 数字在排序数组中出现的次数
## 题目描述
统计一个数字在排序数组中出现的次数。
## 解题思路
### 循环顺序统计
```
# -*- coding:utf-8 -*-
class Solution:
    def GetNumberOfK(self, data, k):
        # write code here
        if len(data) <= 0:
            return 0
        count = 0
        for i in data:
            if i == k:
                count += 1
        return count 
```
