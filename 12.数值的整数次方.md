# 数值的整数次方
## 题目描述
给定一个double类型的浮点数base和int类型的整数exponent。求base的exponent次方。
## 解题思路
### 常规解法
```
# -*- coding:utf-8 -*-
class Solution:
    def Power(self, base, exponent):
        # write code here
        result = 1.0
        if base == 0:                                #考虑分母为0的情况
            return 0
        elif exponent== 0:                           #考虑指数为0的情况
            return 1
        elif exponent < 0:                           #考虑指数为负数的情况
            for i in range(-exponent):
                result *= base
            return 1/result                          
        else:
            for i in range(exponent):
                result *= base
            return result       
```
