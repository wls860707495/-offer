# 调整数组顺序使奇数位于偶数前面
## 题目描述
输入一个整数数组，实现一个函数来调整该数组中数字的顺序，使得所有的奇数位于数组的前半部分，所有的偶数位于数组的后半部分，并保证奇数和奇数，
偶数和偶数之间的相对位置不变。
## 解题思路
### 新建数组
```
# -*- coding:utf-8 -*-
class Solution:
    def reOrderArray(self, array):
        # write code here
        odd,even = [],[]
        for i in array:
            if i&1==1:
                odd.append(i) 
            else:
                even.append(i)
        return odd+even
```
### 冒泡思想
```
# -*- coding:utf-8 -*-
class Solution:
    def reOrderArray(self,array):
        # write code here
        i = 0
        while i<len(array):
            j=len(array)-1
            while j>i:
                if array[j]&1 == 1 and array[j-1]&1 == 0:
                    a = array[j]
                    array[j] = array[j-1]
                    array[j-1] = a
                j -= 1
            i+=1
        return array         
```
