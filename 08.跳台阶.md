# 跳台阶
## 题目描述
一只青蛙一次可以跳上1级台阶，也可以跳上2级。求该青蛙跳上一个n级的台阶总共有多少种跳法（先后次序不同算不同的结果）。
## 解题思路
### 与上题斐波那契数列的递归解决方案类似，也要考虑时间上的性能
```
# -*- coding:utf-8 -*-
class Solution:
    def jumpFloor(self, number):
        # write code here
        num = [1,2]
        if number==0:
            return 0
        if number==1:
            return 1
        if number==2:
            return 2
        else:
            for i in range(number-2):
                num.append(num[-1]+num[-2])
            return num[-1]
```
